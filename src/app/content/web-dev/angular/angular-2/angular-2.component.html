Angular Components

Components are the basic building blocks of an Angular application. Everything you code will be mostly inside a component. Say, you wanna make an About page, you will generate a component that has the content you need in about and in that component itself you will handle the necessary functions needed for a that page.

Let us start by generating a component. To generate a component, use :

            
              ng generate component component-name

Now once you have generated this, you get a component as a folder with 5 parts in it.
