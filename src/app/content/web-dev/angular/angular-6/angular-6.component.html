Angular Directives

Ok so let's start with an example. Assume you have a an element that should be displayed only when the user is logged in. Now what can you do for this? It's as simple as applying an if condition for that element and that's exactly what you can do using these directives. So basically directives are nothing but something that helps us to manipulate the DOM. You can change a DOM element’s appearance, behavior, or layout using these directives. Now there are many of these directives. I will mention a few important ones here and the rest you can look up as you go on building your application.

  • ngFor - For repeating a portion of the HTML template once per each item from an iterable list (Collection).
  • ngIf - To add a condition to add/remove DOM elements.
  • ngClass - This allows us to add or remove CSS classes to an HTML element.

These are just a few that i constantly use and think plays an important role while creating a web application. You use these a few times in your application and number of times you look at ChatGpt will reduce a lot. Atleast for me it was like that XD.
