When you want to generate a component as a child of a paticular module, generate the component in this syntax :

    
      ng generate component module-name/component-name

Angular Interface

Interface in angular is used to create objects. Let's assume we need to make a product with details such as id, name, description, category, technologies etc. We can create an interface for this with the required details and the data type it requires.

To generate an interface, use:

        
          ng generate interface interface-name

Interpolation

Interpolation is a feature there in i think almost every development platform, like its there in react, vanilla js etc. It is a really cool feature that helps to make our application dynamic. There are many other use cases but i feel the most used/ most applicable example is this. For instance, Assume we have an all-product component that has a function called getAllProducts() which gets all the product details from the backend. Now assume we store this function inside products. So right now products have details of all the products.
