Routing in Angular

Now from here on, I assume your are slowly getting the hang of the Angular structure and going further, it's gonna keep getting easier. Once you have got the structure, i recommend start making your own sample-app where you can practice all these things. Then you will have a perfect idea about the structure and it will be much easier going further.

Routing in Angular is actually really simple even though it looks a bit heavy at first. Just a few steps to follow. Consider a component say 'web-dev' you have that you need to be displayed when '/web-dev' is clicked. You can do this in 3 simple steps.

  • Add the routing for that component in its respective routing-module.ts
  • Make sure that exported routing module is there in the module.ts ( here, ApplicationRoutingModule )
  • Now make sure this feature module is there in the main module ie. app.module.ts

Now, you can use '/web-dev' in a-tags or as required to navigate to that component.

What i basically did here was : First, I added the routing for the component in the routing-module.ts inside const routes. This is where we set the name of the routing and stuffy as required and also specify the component that needs to viewed in this syntax:

            
                {{'{path: \'your-path\', component:\'your-component\'},'}}

Once that is done, you just make sure that routing module is there in your feature module and this feature module is imported to your main module as well. Most of this is usually done by default when you generate a module/component but make sure it is there in that required structure.

Also, one other thing that you need to remember is when defining a route, you only specify the route name like {{'{path: \'your-path\', component:\'your-component\'},'}} but when want to use that route, you have to use a forward slash like '/web-dev'. And since Angular is a one page application, we don't use href="/your-route" for component redirection, we use routerLink="/your-route".

See, like i told you, it keeps getting simpler ;)
